Kril is a game about a transgender woman buying girls clothes for the first time. This game is semi-autobiographical, so it mainly reflects my own personal experiences. I designed, programmed, and created all the art assets for the game all on my own. I used Unity and Photoshop to create all the assets. I made this game as my capstone for the NYU Game Design program.
