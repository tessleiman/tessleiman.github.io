Kril is a game about a transgender woman buying gender-affirming clothes for the first time. This game is semi-autobiographical, so it mainly reflects my own personal experiences. I worked as the sole developer on this, providing the design, programming, and art assets. I worked with a musician for the sound track alongside various voice actors.
I showcased this game at Samsung's Pride event and NYU's Summer Game Showcase. I also spoke about this game on the TransGeek panel, discussing the games themes.
I made this game as my capstone for the NYU Game Design program.
